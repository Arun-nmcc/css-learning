Flexbox Cheat Sheet

flex:   ;

- : Defines how much the item will grow relative to others.
  Default: 0 (no growth).
  Examples:
    flex-grow: 0;  /* Does not grow */
    flex-grow: 1;  /* Grows proportionally */
    flex-grow: 2;  /* Grows twice as fast as flex-grow: 1 */

- : Defines how much the item will shrink relative to others.
  Default: 1 (shrinks proportionally).
  Examples:
    flex-shrink: 0;  /* Does not shrink */
    flex-shrink: 1;  /* Default shrinking behavior */
    flex-shrink: 2;  /* Shrinks twice as much as flex-shrink: 1 */

- : Defines the initial size of the item before growth/shrink.
  Default: auto (size based on content).
  Examples:
    flex-basis: 0;     /* No initial size */
    flex-basis: 100px; /* Starts with 100px size */
    flex-basis: auto;  /* Size based on content */

Shorthand Examples:
- flex: 1;      /* Equivalent to flex: 1 1 0 */
- flex: 2;      /* Equivalent to flex: 2 1 0 */
- flex: auto;   /* Equivalent to flex: 1 1 auto */
- flex: none;   /* Equivalent to flex: 0 0 auto */
- flex: 0 1 100px; /* Fixed size with shrinking */
