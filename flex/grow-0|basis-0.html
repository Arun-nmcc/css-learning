Flexbox: Combining flex-grow and flex-basis

This example showcases how different combinations of flex-grow and flex-basis affect the layout of flex items within a container.

Grow: 0, Basis: 0
Grow: 0, Basis: 100px
Grow: 1, Basis: 100px
Grow: 2, Basis: 0

Cheat Sheet

  • flex-grow: Determines how much a flex item will grow relative to other items in the container.
    • flex-grow: 0: The item does not grow.
    • flex-grow: 1: The item grows proportionally to available space.
    • flex-grow: 2: The item grows twice as much as an item with flex-grow: 1.
  • flex-basis: Defines the initial size of the flex item.
    • flex-basis: 0: The item starts with no size, and relies entirely on flex-grow for sizing.
    • flex-basis: 100px: The item starts with an initial width of 100px.
  • Combinations:
    • flex: 1 0 0: Starts with no size, grows proportionally.
    • flex: 0 0 0: Collapses to zero and does not grow.
    • flex: 0 0 100px: Fixed width, does not grow.
    • flex: 1 0 100px: Starts at 100px, grows proportionally.
    • flex: 2 0 0: Starts with no size, grows twice as much as flex: 1.
