CSS Units and Sizing

Font Sizing and Line Height

This text uses a font size of 16px, which is also equivalent to 1rem. The line height is set to 1.6, making it 1.6 times the font size.

Box Width and Height

The container is styled with a width of 50%, a max-width of 600px, and a min-width of 300px. The height is adaptive but constrained between min-height: 100px and max-height: 400px.

Experiment by resizing the window to see how the box behaves!
/* FONT SIZE UNITS: - px: Fixed size (e.g., 16px). - em: Relative to parent font size (1em = same as parent). - rem: Relative to root font size (1rem = root's size, e.g., 16px). - %: Relative to parent size (e.g., 100% = same as parent). */ /* FONT WEIGHT VALUES: - Normal weights: 400 (normal), 700 (bold). - Thin to thick: 100, 200, ... 900. - Keywords: normal, bold, lighter, bolder. */ /* WIDTH/HEIGHT PROPERTIES: - max-width: Sets the maximum width. - min-width: Sets the minimum width. - max-height: Sets the maximum height. - min-height: Sets the minimum height. */
