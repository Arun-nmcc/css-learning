Understanding Deep Nesting with em

Level 1: Parent

The parent has a font size of 18px, and its padding is 1em (18px).

Level 2: Child

The child inherits the parent's font size (18px) and uses `1.5em`, resulting in a computed font size of 27px.

Level 3: Grandchild

The grandchild inherits the child's font size (27px) and uses `0.8em`, resulting in a computed font size of 21.6px.

Level 4: Great-Grandchild

The great-grandchild inherits the grandchild's font size (21.6px) and uses `1.2em`, resulting in a computed font size of 25.92px.
