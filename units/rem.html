Understanding CSS rem

Parent Section

The parent element has a font size of `2rem`, which equals 20px (2 × 10px root size). Padding is `1rem` (10px).

Child Element

The child has a font size of `1.5rem`, which equals 15px. Even though the parent uses `2rem`, this child is still relative to the root size, not the parent.

Grandchild Element

The grandchild has a font size of `1rem`, which equals 10px. It does not cascade with the parent or grandparent; it remains consistent with the root font size.

Independent Section

This section has a font size of `3rem`, which equals 30px. Even though it's outside the parent-child-grandchild structure, its size is still predictable because it is based on the root font size.
