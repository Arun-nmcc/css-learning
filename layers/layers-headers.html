Cheat Sheet

Layer Breakdown

  • Framework: Includes third-party libraries or foundational CSS.
  • Reset: Removes browser defaults for consistent styling.
  • Base: Global styles like font-family, body background, and anchor link styles.
  • Layout: Defines the overall structure, e.g., grids, headers, footers.
  • Components: Reusable UI elements like buttons, cards, modals.
  • Utilities: Small reusable classes for margin, padding, alignment.

Key Points

  1. Order Matters: Later layers override earlier ones if they have the same specificity.
  2. Utility First: Utility classes are applied last, allowing quick overrides.
  3. Scalability: This structure makes CSS maintainable and extensible.
