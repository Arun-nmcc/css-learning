CSS Positioning: Relative

The container itself has position: relative. Each box inside demonstrates the behavior of position: relative.

Box 1: I have position: relative with top: 20px and left: 20px. I reserve space in my original position but appear offset. Notice I overlap with Box 2!

Box 2: I also have position: relative with top: -20px and left: 40px. I overlap Box 1 because of stacking context and a higher z-index.

Key Points:

  • position: relative allows the element to maintain its original space but appear shifted using top, left, right, or bottom.
  • This creates a **stacking context**, meaning elements can overlap each other using z-index.
  • Other elements are not aware of the visual offset; they treat the element as if it were in its original position.

Tip: Use position: relative when you need an element offset but still want it to interact with other elements in the document flow.
