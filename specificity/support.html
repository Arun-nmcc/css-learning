CSS Specificity: vh vs svh

Difference between vh and svh:
- vh stands for "Viewport Height" and represents 1% of the height of the viewport, including areas hidden by browser interfaces like the address bar.
- svh (Small Viewport Height) represents 1% of the height of the **visual viewport**, excluding the browser UI such as the address bar. It adapts better in modern mobile browsers when the UI resizes dynamically.

Fallback:
If a browser does not support svh, we can use vh as a fallback using @supports.
