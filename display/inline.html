Inline Elements Cheat Sheet

Inline elements do not break onto a new line and only take up as much width as their content. They are commonly used for styling small portions of text or inline graphics.

Key Characteristics

Below are some important behaviors of inline elements with visual demonstrations:

  • Height: Inline elements ignore the height property. Height set to 50px (ignored).
  • Margin: Margins generally have no effect on inline elements. Margin set to 10px (ignored).
  • Padding: Padding works but doesn't push neighboring elements away. Padding set to 20px.
  • Font Size: Changing font size alters the content area but may break consistency. Font size set to 24px.

Inline Elements and Block Elements

Inline elements like <span>, <a>, and <strong> should be used for styling or adding interactivity within a line of text. For structural layout, use block or inline-block elements instead.
