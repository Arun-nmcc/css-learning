CSS Grid Layout Example

Cheatsheet:

  • display: grid: Enables the grid layout.
  • gap: Adds spacing between items without affecting outer margins.
  • grid-template-columns: Defines the number and size of columns.
  • grid-template-rows: Defines the number and size of rows.
  • grid-column-start / grid-column-end: Defines the starting and ending column lines.
  • grid-row-start / grid-row-end: Defines the starting and ending row lines.
  • Implicit grid: Rows or columns are created automatically when needed.
  • Explicit grid: Rows or columns are explicitly defined.
  • Shorthand: Use grid-column and grid-row to combine start/end or start/span.
